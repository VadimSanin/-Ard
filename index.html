Шкаф для проращивания зерновой культуры

Система управления с помощью Arduino

О проекте

Этот шкаф предназначен для проращивания зерновых культур с использованием автоматизированной системы управления на базе Arduino. Он поддерживает оптимальные условия для роста: температуру, влажность и освещение.

Технические детали

  • ESP8266 (Wi-Fi модуль)

  • Дисплей

  • Вентилятор

  • Нагревательный элемент

  • Датчик температуры и влажности

  • Светодиодные лампы
